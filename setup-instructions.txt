# TauMail EC2 Setup Instructions

## Instance Details:
- Instance ID: i-02213e9e03a1fbe0f
- Public IP: 54.224.231.238
- Key Pair: taumail-key.pem

## Next Steps:

1. SSH into the instance:
   ssh -i taumail-key.pem ec2-user@54.224.231.238

2. Install Docker and setup:
   sudo yum update -y
   sudo yum install -y docker git
   sudo systemctl start docker
   sudo systemctl enable docker
   sudo usermod -a -G docker ec2-user

3. Install Docker Compose:
   sudo curl -L "https://github.com/docker/compose/releases/download/v2.20.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
   sudo chmod +x /usr/local/bin/docker-compose

4. Upload deployment package:
   scp -i taumail-key.pem infrastructure/taumail-deployment.tar.gz ec2-user@54.224.231.238:~/

5. Extract and setup:
   mkdir -p /opt/taumail
   cd /opt/taumail
   tar -xzf ~/taumail-deployment.tar.gz
   docker-compose up -d

6. Test web interface:
   http://54.224.231.238:3001

7. Add DNS records from dns-config-simple.txt
